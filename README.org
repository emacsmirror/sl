* sl.el

[[file:sl.gif]]

sl.el is an Emacs clone of [[https://github.com/mtoyoda/sl][sl(1)]].

** Usage

Type =M-x sl= or =M-x sl-little=.

Currently, sl(1) with no option and with -l option are implemented, other
options (i.e., -aFc) are not supported.
